# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML5** and **CSS3**. This project showcases my profile, technical skills, education, and contact information through a clean and minimal user interface.

## ✨ Features

- Responsive and modern layout
- Smooth single-page navigation
- Hero section with introduction
- About Me section
- Technical Skills showcase
- Education timeline
- Contact form UI
- Social media links (LinkedIn, GitHub, Twitter)
- Clean and organized code structure

## 🛠️ Built With

- HTML5
- CSS3
- Google Fonts (Plus Jakarta Sans)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Navigate to the project directory

```bash
cd portfolio
```

3. Open `index.html` in your browser.

No additional setup or dependencies are required.
